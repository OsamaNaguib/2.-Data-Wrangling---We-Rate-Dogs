Introduction¶

This is project is requested during Udacity data analysis course. The objective of the WeRateDogs project is to anlyze twitter data regarding rating of dogs.

Data comes from 3 different sources and with 3 different formats:¶

  • Programmatic Download of image-prediciton.tsv
  • Manual Download of twitter-archive-enhanced.csv
  • Querying Twitter API and Composing json to txt file tweet_json.txt

The Image Prediction data was supplied from a software based on Macihne learnig, and were required to be downloaded programatically then to be used as a validation tool thorugh the pictures of the dogs, which some didn't have and were eliminated from the master data.

On the other hand the twitter archived was supplied for manual download through Udacity servers, and it is on CSV format, and serves as the source of the tweet_id, retweet and favourite counts, which had to be filtered to get the orignal tweet count. It also served as the source for the dogs stages data, which required very interesting cleaning approaches too.

The third part was suplied as quering Twitter API through a developer Twitter account, which I gained though some long process/investigation and a lot of communications/emails till I got the approval. then I was able to querying the Twitter API to build a josn file line by line seprated by empty line, then I iterate line by line though json.dumb till forming the TXT file, which formed the third dataset for the analysis.

Tweepy, matplotlib, pandas, numpy and other packages were used in this project.

Using the learned systematic approach for data wrangling, passing through gathering, assessing, cleaning of data, then to store it in a new clean master data in csv format, where the anlysis and visualization were done effectively, and it is the main objective of this report

This report shall answer the following questions:

  • What is the most common dog breed in this dataset?
  • Which dog breed received higher favourites and tweet count?
  • Which dog breed had the highest rating (numerator) among other dog breeds?
  • Which dog breed received the highest retweet and favourites counts?
  • Which dog stage received higher rating?
  • Which dog stage had higher prediction confidence in image prediction data?
  • What is the most common dog name in this dataset?
  • Is there is a correlation between retweet count and rating_numerator?
